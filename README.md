# DNA Sequence Analyzer

This is my first bioinformatics project using Python.

# Features
- Calculates sequence length
- Counts A, T, G, C nucleotides
- Computes GC content

# Tech Used
- Python

# Purpose
I built this project as part of my journey into bioinformatics to apply programming to biological data analysis.

# Example : ATGCGCGTA
Output
Sequence Length: 9  
A: 2  
T: 2  
G: 3  
C: 2  
GC Content: 55.56%
